# JASI Smart Decor

**JASI Smart Decor** is a React/Tailwind application showcasing modern furniture collections and interior design inspiration for customers in Morocco.

## Local development

To run the project locally you only need Node.js (LTS) and npm or yarn installed.

```sh
# clone repository
git clone <YOUR_GIT_URL>
cd jasi-decor-studio-main

# install dependencies
npm install   # or yarn

# start development server
npm run dev
```

The app will be available at `http://localhost:5173` by default.

## Build & deploy

Use `npm run build` to generate a production bundle in the `dist/` folder. Deploy that output to any static‑hosting service (Vercel, Netlify, GitHub Pages, etc.).

## Features & technologies

- **React 18** with TypeScript
- **Vite** as build tool
- **Tailwind CSS** plus shadcn‑ui components
- Responsive layout and SEO‑friendly markup

## Updating content

Modify pages under `src/pages` and components in `src/components` as needed. Run `npm run lint` to check formatting and `npm run test` for unit tests.

## SEO optimization

Meta tags and Open Graph data are defined in `index.html`; make sure to keep them up to date with your site title, description, and canonical URL.

Sitemap and robots directives live in `public/robots.txt`.
